# Components Directory

This directory contains all React components for the Cloud + AI Assessment Landing Page.

## Structure

Components will be organized by section:

- `HeroSection.tsx` - Hero section with headline and CTAs
- `ProblemSection.tsx` - Problem statement and pain points
- `SolutionSection.tsx` - Assessment modules overview
- `DeliverablesSection.tsx` - What clients receive
- `ProcessSection.tsx` - 4-step workflow
- `OutcomesSection.tsx` - Expected business outcomes
- `TargetAudienceSection.tsx` - Who this is for
- `DifferentiatorsSection.tsx` - Why choose us
- `CaseStudiesSection.tsx` - Success stories
- `FAQSection.tsx` - Frequently asked questions
- `BottomCTASection.tsx` - Final call-to-action

## Shared Components

- `Button.tsx` - Reusable button component
- `Card.tsx` - Reusable card component
- `Section.tsx` - Section container wrapper
